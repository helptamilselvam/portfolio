# Tamilselvam Jeyaraman | Junior Data Engineer Portfolio

## Quick Start

1. Double-click START.bat
2. Browser opens automatically at http://localhost:3000
3. Done!

## Files
- index.html    → Portfolio (also works standalone)
- server.js     → Backend server
- database.js   → SQLite database
- package.json  → Dependencies
- START.bat     → Windows launcher
- vercel.json   → Vercel deploy config

## Deploy to Vercel
1. Push all files to GitHub
2. Go to vercel.com → Import repo → Deploy
3. Live at https://yourname.vercel.app
